# Convenciones de Escritura

* *nombre de archivo*

## npm

* *npm* debe estar en minúsculas aunque se encuentre al comienzo de una frase.

Uso

```bash
npm i alt --save
```

¡Recuerda incluir `--save` o `--save-dev` siempre que instales algo con *npm*!
